# GitHub Feedback

StorySeed Classroom is local-first and does not collect telemetry. The best feedback is a short GitHub issue with enough detail to reproduce what happened.

## Helpful Bug Reports

Please include:

- Windows version, if you know it.
- Python version, if StorySeed showed it or `python --version` works.
- Whether you used the release ZIP or a git clone.
- What you clicked just before the problem happened.
- A screenshot, if the page looked wrong.
- The prompt seed, mode, age band, subject, and creativity value if the issue was about a generated prompt.

## Useful Prompt Feedback

Good reports are specific:

- "This was too young for age 11-14."
- "The Review page said ready, but the task still felt vague."
- "The worksheet exported, but the teacher note was not useful."
- "This seed made something funny or surprisingly good."

## Feature Requests

Please keep feature requests tied to the app's promise:

- local-first
- no accounts
- no API keys
- no cloud calls
- teacher-safe by default
- one page, one job, one sensible next step

Large AI integrations, DOCX import, and packaged installers are possible future work, but the current release should stay simple enough for someone to unzip and run.
